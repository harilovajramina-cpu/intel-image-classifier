# Intel Image Classifier — Julianna

A deep learning web application for image classification using **PyTorch** and **TensorFlow**, deployed with **Flask**.

The model classifies images into 6 categories:
- Buildings 🏙️
- Forest 🌲
- Glacier 🧊
- Mountain ⛰️
- Sea 🌊
- Street 🛣️

---

## 🚀 Project Features

- Dual framework support (PyTorch + TensorFlow)
- CNN-based image classification
- Flask web interface
- Image upload & real-time prediction
- Confidence score visualization
- Emoji-based results
